To lock this room, add two environment variables in Vercel for the Tokenstopia project, then redeploy.

1. Add OPS_PASSWORD

This is the password you will type on your phone or desktop to enter the room.

2. Add OPS_SESSION_SECRET

Use a long random string. It signs the private session cookie so the room stays protected.

3. Redeploy once

After redeploy, refresh this page and you will see the login form instead of this setup note.
